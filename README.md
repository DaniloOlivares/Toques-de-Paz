
# Sistema de gestión masajes Toques de Paz.
## Descripción
Sistema web que permite administrar, reservar masajes, clientes, promociones y agendas de masoterapeutas.

## Objetivo
Digitalizar el proceso de reservas y administración del negocio.

## Tecnologías (planificadas)

- React
- Node.js
- Express
- PostgreSQL
- Docker
